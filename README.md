# MemberLog 🏋️

헬스장 회원 관리 앱 — React + Vite

## 기능
- 회원 등록 (이름, 연령, 성별, 운동목적 다중선택, 건강 특이사항)
- 날짜별 건강 기록 이어쓰기 (타임라인)
- 회원 폴더 / 만료 회원 폴더 분리 관리
- 검색 기능

---

## 🚀 Vercel 배포 방법 (5분)

### 1단계 — GitHub에 올리기
```bash
# 이 폴더를 GitHub에 새 레포지토리로 업로드
# github.com → New repository → "memberlog" 이름으로 생성
# 파일 전체 업로드 (또는 git push)
```

### 2단계 — Vercel 연결
1. [vercel.com](https://vercel.com) 접속
2. **GitHub으로 로그인**
3. **"Add New Project"** 클릭
4. memberlog 레포지토리 선택
5. **Deploy** 클릭 → 자동 빌드!

### 3단계 — 폰 홈화면에 추가
1. 배포된 URL을 폰 Safari(iOS) 또는 Chrome(안드로이드)으로 열기
2. **공유 버튼 → "홈 화면에 추가"**
3. 앱 아이콘으로 설치 완료! 🎉
